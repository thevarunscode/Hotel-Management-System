# LuxeStay - Hotel Management System

A complete web-based Hotel Management application.

## Features
- Dashboard with real-time stats
- Room & Room Type Management
- Guest Directory
- Reservations & Check-in/Check-out Workflows
- Billing & Invoicing System

## Architecture
- **Frontend**: Vanilla HTML/CSS/JavaScript. It makes asynchronous `fetch` calls to the local API.
- **Backend**: Node.js & Express. Provides REST API endpoints.
- **Database**: SQLite3.

## Setup & Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open your browser to `http://localhost:3000`
